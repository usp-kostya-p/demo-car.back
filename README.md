## Installation

```bash
$ npm install
```
## Migrations and seeds

```bash
$ npm run migration:run
$ npm run seed
```


## Running the app

```bash
# up database container
$ docker-compose up

# stop database container
$ docker-compose down -v

# run server
$ nest start

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
